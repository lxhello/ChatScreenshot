# ChatExtractor-Screenshot Version 使用指南

## 快速开始

### 1. 准备工作

**手机端：**
- 开启"开发者选项"（设置 → 关于手机 → 连续点击版本号7次）
- 开启"USB调试"（开发者选项 → USB调试）
- 连接电脑后授权"USB调试"弹窗

**电脑端：**
```bash
# 检查 ADB 是否安装
adb version

# 检查设备连接
adb devices
# 应显示：17211JECB14586    device
```

### 2. 运行程序

```bash
cd ChatExtractor-Screenshot
start.bat

# 开发环境也可以直接运行
python3 main.py
```

### 3. 操作流程

**初始化：**
1. 程序自动检测设备和当前 App
2. 确认 App 名称（可修改）
3. 创建项目目录

**截图采集：**
1. 选择"1. 自动滚动截图"
2. 在手机上打开目标聊天页面
3. 按回车继续
4. 确认聊天标题（可修改）
5. 按回车开始自动截图

**流程说明：**
- 第一步：自动滚动到聊天最顶端（最早的消息）
- 第二步：从顶部开始向下逐屏截图
- 第三步：检测到重复画面自动停止

## 配置调整

### 根据手机分辨率调整滚动参数

**1080x2400 (默认):**
```python
"swipe_start": [540, 1800],
"swipe_end": [540, 600],
```

**1080x1920:**
```python
"swipe_start": [540, 1500],
"swipe_end": [540, 500],
```

**1440x2560:**
```python
"swipe_start": [720, 2000],
"swipe_end": [720, 700],
```

### 调整滚动速度

```python
# 加快速度（可能遗漏）
"swipe_interval": 1.0,

# 减慢速度（更稳定）
"swipe_interval": 2.0,
```

### 调整重复检测

```python
# 更敏感（容易误判到顶）
"duplicate_threshold": 2,

# 更宽松（避免误判）
"duplicate_threshold": 5,
```

## 故障排查

### 问题1：滚动到顶部时间过长

**原因：** 聊天记录太多，滚动次数多

**解决：**
1. 手动先滚动到接近顶部
2. 减小 `swipe_interval` 参数
3. 或按 Ctrl+C 中断，从当前位置开始截图

### 问题2：截图有遗漏

**原因：** 滚动距离太大，跳过了部分内容

**解决：**
```python
# 减小滚动距离
"swipe_start": [540, 1600],  # 原 1800
"swipe_end": [540, 700],     # 原 600
```

### 问题3：截图有重复

**原因：** 滚动距离太小，重复截取相同区域

**解决：**
```python
# 增大滚动距离
"swipe_start": [540, 2000],  # 原 1800
"swipe_end": [540, 500],     # 原 600
```

### 问题4：无法识别聊天标题

**原因：** 不同 App 界面结构不同

**解决：**
- 程序会提示手动输入
- 直接输入对方昵称或群名
- 后续可在文件夹中重命名

## 高级技巧

### 技巧1：批量采集多个聊天

```bash
1. 运行程序
2. 选择"1. 自动滚动截图"
3. 完成第一个聊天
4. 选择"3. 切换聊天"
5. 在手机上切换到下一个聊天
6. 重复步骤2-5
```

### 技巧2：中断后继续

```bash
# 如果中途中断，可以：
1. 重新运行程序
2. 选择同一个 App（会使用已有项目目录）
3. 打开同一个聊天
4. 输入不同的标题（如"张三_续"）
5. 继续截图
```

### 技巧3：自定义输出目录

```python
# 修改 config.py
"output_root": "/path/to/your/backup",
```

## 输出文件说明

### 截图文件

- **命名规则：** `聊天标题_序号.png`
- **序号：** 从 001 开始，按时间顺序递增
- **格式：** PNG（无损压缩）
- **分辨率：** 与手机屏幕一致

### 日志文件

```
logs/capture_log.txt

[2026-04-09 10:30:15] 聊天: 张三, 截图数: 25
[2026-04-09 10:35:42] 聊天: 工作群, 截图数: 48
```
